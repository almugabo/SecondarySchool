**Exercise 2** ‚ÄĒ hypotenuse = $2\sqrt{2}$, one leg = 2

Find the other leg by Pythagoras:
$$(2\sqrt{2})^2 = 2^2 + x^2 \implies 8 = 4 + x^2 \implies x = 2$$

Both legs are equal to 2 ‚ÄĒ this is an **isosceles right triangle** with both acute angles = 45¬į.

For the angle $\theta$ opposite the first leg:
$$\sin(\theta) = \frac{2}{2\sqrt{2}} = \frac{1}{\sqrt{2}} = \boxed{\frac{\sqrt{2}}{2}}$$
$$\cos(\theta) = \frac{2}{2\sqrt{2}} = \frac{1}{\sqrt{2}} = \boxed{\frac{\sqrt{2}}{2}}$$
$$\tan(\theta) = \frac{2}{2} = \boxed{1}$$

Both acute angles give the same values since the triangle is isosceles.

**Exercise 4** ‚ÄĒ Co-function identity: $\sin(\theta) = \cos(90¬į - \theta)$

For $\theta = 30¬į$:
$$\sin(30¬į) = \frac{1}{2} \quad \text{and} \quad \cos(90¬į - 30¬į) = \cos(60¬į) = \frac{1}{2} \checkmark$$

For $\theta = 45¬į$:
$$\sin(45¬į) = \frac{\sqrt{2}}{2} \quad \text{and} \quad \cos(90¬į - 45¬į) = \cos(45¬į) = \frac{\sqrt{2}}{2} \checkmark$$

In both cases the identity holds. This works because in a right triangle, the two acute angles are complementary, so the opposite side for one angle is the adjacent side for the other.

### D. Trigonometric Functions & Graphs

**Exercise 1** ‚ÄĒ $y = 6\sin\!\left(\dfrac{\pi}{3}\,x\right) - 1$

- **Period** = $\dfrac{2\pi}{\pi/3} = \boxed{6}$ (6 units, not degrees ‚ÄĒ $x$ is in real numbers here)
- **Vertical shift** = $\boxed{-1}$ (down 1 unit)

**Exercise 2** ‚ÄĒ $y = 3\sin(x) + 1$

| $x$ | $0¬į$ | $90¬į$ | $180¬į$ | $270¬į$ | $360¬į$ |
|---|---|---|---|---|---|
| $y$ | $1$ | $4$ | $1$ | $-2$ | $1$ |

- **Minimum** = $-3 + 1 = -2$ at $x = 270¬į$
- **Midline**: $y = 1$

**Exercise 3** ‚ÄĒ $h = 3\sin\!\left(\dfrac{\pi}{4}\,t\right) + 8$

a) Maximum and minimum heights:
$$h_{\max} = 8 + 3 = \boxed{11 \text{ m}}$$
$$h_{\min} = 8 - 3 = \boxed{5 \text{ m}}$$

b) Period:
$$T = \frac{2\pi}{\pi/4} = \boxed{8 \text{ seconds}}$$

c) When $h = 8$:
$$3\sin\!\left(\frac{\pi}{4}\,t\right) + 8 = 8 \implies \sin\!\left(\frac{\pi}{4}\,t\right) = 0$$
$$\frac{\pi}{4}\,t = 0,\ \pi,\ 2\pi,\ 3\pi,\ 4\pi$$
$$t = 0,\ 4,\ 8,\ 12,\ 16 \text{ s}$$
$$\boxed{t = 0,\ 4,\ 8,\ 12,\ 16 \text{ seconds}}$$

**Exercise 4** ‚ÄĒ Comparing $y = 2\sin(x)$ and $y = \sin(2x)$

a) Differences:

| Property | $y = 2\sin(x)$ | $y = \sin(2x)$ |
|---|---|---|
| Amplitude | **2** | 1 |
| Period | 360¬į | **180¬į** |
| Midline | 0 | 0 |

Student A's graph is **taller** (amplitude 2); Student B's graph is **narrower** (twice as fast).

b) At $x = 90¬į$:
- Student A: $2\sin(90¬į) = 2 \times 1 = \mathbf{2}$
- Student B: $\sin(2 \times 90¬į) = \sin(180¬į) = \mathbf{0}$

**Student A's function has the greater value** ($2 > 0$).
